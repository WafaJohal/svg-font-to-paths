Convert SVG Fonts to Paths

Convert monospace fonts in an SVG to a SPRITE2 or ZIP file containing an SVG for every glyph that you choose. Right now this tool is only designed for monospace fonts.

The SPRITE2 file can be imported into Scratch. It has a costume for every successfully converted character and `advx` and `advy` variables that tell how wide and tall each character is.

The ZIP file contains an SVG for each character, named for the character they contain, like "asterisk.svg", "lowercase/a.svg", and "uppercase/A.svg".

To convert a font file(.ttf, .dfont, etc.) to an SVG font file(.svg), try a different web-based tool like Online Font Converter.
