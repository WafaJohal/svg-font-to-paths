Convert SVG Fonts to Paths

Choose a set of characters and an SVG file containing a font and convert it to a .sprite2 file to import into Scratch with a costume for every character or a .zip file containing an SVG for every character that can be imported individually.

A .sprite2 file is a zip archive containing numbered SVGs and a "sprite.json" file. When imported into Scratch, it also has a "advx" variable that says how many steps to move forward from the start of one character to another when the size is 100% so that they don't overlap.

The .zip contains an individual SVG for each character, named like "asterisk.svg" or "lowercase/a.svg" or "uppercase/A.svg". They can also be imported into Scratch individually, or into existing costumes.

To convert a font file(.ttf, .dfont, etc.) to an SVG font file(.svg), try a different web-based tool like Online Font Converter.
